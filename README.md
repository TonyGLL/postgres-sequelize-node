# Nodejs API

### Tools:
 - postgresql 
 - typescript
 - nodejs
 - expressjs
 - docker

### Try it
```
git clone https://github.com/TonyGLL/postgres-sequelize-node.git
cd postgres-sequelize-node
npm i
npm run dev
```

### Serve with docker (api + postgresql images) 
```
docker-compose up -d --build
```
